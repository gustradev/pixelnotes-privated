=========================================================
MASTERPLAN SUMMARY
=========================================================

Pixel Notes adalah aplikasi notes dengan secret chat mode.
Arsitektur dirancang tunnel-only, ephemeral, dan security-first.

Core goals:
- Realtime secret chat
- Ephemeral storage Redis TTL 86400
- Dual unlock (password + biometric)
- Face verification orchestration (GPU service)
- Remote flush real-time
- Stealth notification signaling

=========================================================
CURRENT IMPLEMENTATION BASELINE
=========================================================

Frontend:
- Flutter Web/Android/iOS
- Notes surface UI + secret login + secret chat UI
- WebSocket client + biometric service

Backend:
- FastAPI auth/chat/notes/flush/websocket/face routes
- JWT auth + rate limiting + replay protection
- Security headers + sanitized logging

Data layer:
- Redis ephemeral storage
- TTL chat key enforcement

Face service:
- Dedicated microservice (`face-gpu-service`)
- Health + verify endpoints

Infra:
- docker-compose base + dev/staging/prod overlays
- Cloudflare tunnel config template
- Monitoring overlay (Prometheus + Grafana)

Quality:
- Backend tests
- Flutter analyze
- GitHub Actions CI
